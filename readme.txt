Vemetric is a lightweight, privacy-first analytics tool that helps you understand how your users are interacting with your website.

== Description ==
This is the official WordPress plugin for Vemetric, that automatically adds Vemetric to your website and helps you understand how your users are interacting with your website.

**Features:**
- Automatically injects the Vemetric script and tracks pageviews and outbound link clicks.
- Optionally track custom events via data attributes or JavaScript.
- Ability to track events in the PHP backend via the Vemetric PHP SDK.

== Installation ==
1. Install the plugin through the WordPress Plugins page, or upload the plugin files to the `/wp-content/plugins/vemetric` directory.
2. Activate the plugin through the 'Plugins' page in WordPress.
3. Provide your Vemetric project token in the plugin settings. (You can find it in your Vemetric project settings)
4. You're ready to track your website's traffic and user interactions. Vemetric automatically adds the script to your website.

== Frequently Asked Questions ==

= What kind of Analytics does Vemetric provide? =
Vemetric provides you with Web- and Product Analytics. Doesn't matter if you're running a simple website or a more complex product like a SaaS, Vemetric has you covered and you don't need another tracking solution in your toolbelt.

= Does Vemetric use Cookies? =
By default Vemetric doesn't use any kind of Cookies and all our servers are hosted in the EU. Vemetric is therefore GDPR compliant. For identifying recurring visitors on the same day, we generate a unique hash for each of your visitors with a daily rotating salt.

= Can I see a demo of Vemetric in action? =
Yes, you can take a look at a [public dashboard with real live data](https://app.vemetric.com/public/snappify.com)

= Is Vemetric Open Source? =
Yes, Vemetric is open source and licensed under the AGPLv3 license. You can find the code on [GitHub](https://github.com/vemetric/vemetric).
